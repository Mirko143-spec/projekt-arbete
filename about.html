About this project

This quiz app is built with vanilla HTML, CSS and JavaScript as a school project. It uses the Open Trivia API, includes a countdown timer, a points system, and now a persistent high-score list using localStorage.

You can use it as a base to add more features in the future, such as:

  • Multiple quiz topics
  • More visual feedback and animations
  • Sharing scores with friends
